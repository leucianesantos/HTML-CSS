MEU DESAFIO 2 - Lista de definições

É como se fosse um dicionário, temos os termos e as suas descrições. É uma lista sem demarcadores, mas bem útil em alguns casos. Toda lista de definições está dentro de uma tag dl---/dl (definition list) Cada termo é um tag dt (definition term) e cada descrição é um dd (definition description). Assim como os itens da lista, essas duas últimas tags possuem fechamento opcional, segundo a referência oficial da HTML5.

Habilidade:
Habilidade é o substantivo feminino que indica a qualidade de uma pessoa hábil, que revela capacidade para fazer alguma coisa.
Humildade:
Humildade é a qualidade de quem age com simplicidade, uma característica das pessoas que sabem assumir as suas responsabilidades, sem arrogância, prepotência ou soberba.

Bondade:
Bondade é uma característica daquele que tem inclinação para o bem. Consiste numa qualidade das pessoas que praticam boas ações.
Reconhecimento:
Conhece-se pelo nome de reconhecimento a ação e o efeito de reconhecer. O verbo refere-se ao ato de examinar algo ou alguém com atenção, registar algo para conhecer o seu conteúdo, confessar alguma situação ou aceitar um novo estado das coisas.
